# kubernetes-alura
Projetos da formação alura de orquestração de containers

Uso de vários recursos do kubernates, como Services, Pods, Deployments, StatefullSets, Volumes, PersistentVolume, PersistentVolumeClaim, Jobs, ConfigMap, HorizontalPodAutoscaler
